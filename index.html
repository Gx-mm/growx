GrowX Panel – Complete User Guide

Welcome to GrowX – India's Most Affordable SMM Platform यह guide आपको GrowX panel का सही तरीके से उपयोग करना सिखाएगा।

1. Account Login / Create Account Guide

GrowX panel उपयोग करने के लिए आपको पहले account बनाना होगा।

Create New Account

  1. Create New Account पर क्लिक करें
  2. अपना Username डालें
  3. Gmail Address डालें
  4. Mobile Number डालें
  5. एक strong Password बनाएं
  6. Create Account पर क्लिक करें

Account create होने के बाद आप login कर सकते हैं।

Login Guide

  1. Login page पर जाएं
  2. अपना Username डालें
  3. Password डालें
  4. Login button पर क्लिक करें

Login होने के बाद आपका GrowX Dashboard open हो जाएगा।

2. Home Page (Dashboard) Guide

Login करने के बाद आपको GrowX Dashboard दिखाई देगा।

  • Welcome Message
  • User ID
  • Account Verification Status
  • Available Balance

Dashboard Buttons

Wallet – यहां से आप अपने account में balance add कर सकते हैं।

New Order – यहां से आप Instagram services order कर सकते हैं।

History – यहां आपके सभी orders की history दिखाई देगी।

यह page Instagram services order करने के लिए होता है।
